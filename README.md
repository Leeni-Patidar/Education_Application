# 📚 Education Application

An interactive **Education Platform** built using the MERN stack that helps students learn, manage study materials, and track their progress efficiently.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register with JWT)
- 👨‍🎓 Student Dashboard
- 📚 Course / Notes Management
- 📝 Quiz
- 📊 Progress Tracking
- 🛠 Admin Panel (Manage Users & Content)
- 📱 Fully Responsive Design

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Tools & Technologies
- JWT Authentication
- REST API
- Git & GitHub
